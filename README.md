# 孙哥大脑 · Sun Ge Brain

用**孙宇晨（Justin Sun）的 28 个思考框架和 18 条趋势判断**分析任何问题。

冷眼拆解，不吹不黑。学他看问题的方式，不是学他做人。

## 安装

```
/plugin install github.com/<your-username>/sun-ge-brain
```

或手动：`cp -r .claude/skills/sun-ge-brain ~/.claude/skills/`

## 使用

```
/孙哥大脑 帮我分析一下 XX 项目靠不靠谱
```

自然语言也能触发："用孙哥大脑分析""孙宇晨视角怎么看""拆解一下 XX"

## 四种模式

| 模式 | 场景 | 示例 |
|------|------|------|
| **A 快速诊断** | 项目/决策靠不靠谱 | "帮我看看这个项目" |
| **B 深度蒸馏** | 拆解一个人/公司 | "拆解一下XX" |
| **C 趋势判断** | 方向前景 | "Agent支付赛道怎么看" |
| **D 孙哥推演** | 模拟孙宇晨怎么做 | "如果孙宇晨做AI" |
